# 电信用户流失预测项目（Telco Customer Churn Prediction）

## 1\. 项目简介

本项目为**电信行业用户流失二分类预测**实战，基于公开电信用户数据集，完成从**数据清洗、探索性分析、特征工程到多模型建模评估**的全流程数据分析，定位高流失风险用户特征，为企业用户留存提供数据依据。

## 2\. 数据集信息

- 名称：Telco\-Customer\-Churn

- 规模：7043 条用户记录，21 个特征

- 目标变量：`Churn`（Yes = 流失，No = 未流失）

- 核心字段：用户属性、在网时长、服务开通情况、合约类型、费用信息、支付方式

## 3\. 技术栈

- 数据处理：Pandas、NumPy

- 可视化：Matplotlib、Seaborn

- 机器学习：Scikit\-learn、XGBoost

- 分析工具：Jupyter Notebook

## 4\. 项目流程

### 4\.1 数据加载与探查

读取数据集，查看字段类型、基础统计量，检查缺失值与重复值，识别数据质量问题。

### 4\.2 数据清洗

- 类型转换：将`TotalCharges`转为数值型，删除转换后缺失样本

- 冗余剔除：删除无业务意义的`customerID`字段

- 值域统一：将`No internet service`/`No phone service`统一替换为`No`

- 去重处理：清除重复记录，保证数据唯一性

### 4\.3 探索性数据分析（EDA）

- 整体流失分布：样本流失率约**26\.6%**

- 用户画像：老年用户、无家属、短在网时长用户流失率更高

- 服务特征：光纤用户、未开通安全 / 技术支持用户流失集中

- 合约与费用：按月合约流失率远高于年付合约；月费 70–110 元为流失高发区间

- 相关性分析：在网时长、合约类型、月均费用与流失相关性显著

### 4\.4 特征工程

- 类别特征：One\-Hot Encoding 独热编码

- 数值特征：`MinMaxScaler`归一化`tenure`/`MonthlyCharges`/`TotalCharges`

- 特征筛选：保留有效建模特征，剔除无关变量

### 4\.5 模型构建与评估

- 数据划分：训练集：测试集 = 8:2

- 模型选型：岭分类器、逻辑回归、决策树、随机森林、GBDT、XGBoost 等

- 评估指标：准确率、AUC\-ROC、精确率、召回率、F1、分类报告

- 基准模型：以岭分类器完成训练、预测与效果验证

## 5\. 核心业务发现

1. 合约类型：**按月合约用户流失率（57%）** 远高于 1 年（11%）、2 年（3%）合约

2. 在网时长：在网时长越短，流失风险越高，新用户为高危群体

3. 费用水平：月费 70–110 元中高资费用户流失更集中

4. 增值服务：未开通在线安全、技术支持的用户流失率显著更高
